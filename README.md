# 🚀 Startup Due Diligence Toolkit

Hey there! Thanks for checking out my repository.

I built this **Startup Due Diligence Toolkit** as a personal project. I've always been fascinated by venture capital and wanted to see if I could build a real-time dashboard that actually helps automate the early stages of market research and deal flow tracking.

If you're in VC or IB, you know how tedious it can be to manually track which sectors are heating up, who is leading what round, and what the general sentiment is around emerging tech. I built this toolkit to solve exactly that.

## What It Actually Does

Instead of staring at spreadsheets, this dashboard gives you:
*   **Sector Funding Velocity:** A quick chart showing exactly where the money is flowing right now (e.g., is Generative AI still hot, or is Climate Tech catching up?).
*   **Live Deal Flow Feed:** A tracker of who is investing in what.
*   **NLP Sentiment Analysis:** I built a Python engine that reads recent news headlines and spits out an overall "market sentiment" score using TextBlob. It's super cool to see how news correlates with funding!
*   **Emerging Topics:** Line charts tracking how often specific industries are mentioned over time.

## How I Built It

I wanted everything to be extremely fast and look beautiful, so I didn't use any generic component libraries.

*   **Frontend:** React and Vite. I wrote all the CSS from scratch using a glassmorphism design system because I wanted the dashboard to look premium and sleek.
*   **Data Vis:** Recharts.
*   **Backend Engine (Python):** In the `engine/` folder, there are a couple of scripts I wrote using `BeautifulSoup4` (for scraping) and `TextBlob` (for sentiment analysis). Right now, the React app uses mock data generated by these scripts so it runs instantly without hitting API limits.

## Want to run it locally?

It's super easy to spin up. You just need Node.js.

```bash
# Clone it and install dependencies
npm install

# Start the local server
npm run dev
```

If you want to play around with the Python scraping/sentiment scripts:
```bash
cd engine
pip install beautifulsoup4 textblob requests

# Test the sentiment gauge logic
python sentiment_analyzer.py
```
